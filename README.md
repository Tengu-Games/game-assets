# Game Assets

Asset repository for [dojos.fun](https://dojos.fun) — sprites, images, sounds, and other game resources.

## Structure

- `dialog/` — UI panels, tabs, dialogs
- `scenes/` — Scene backgrounds and imagery
- Other asset folders as needed

## Usage

Assets are consumed by the main game client. Clone this repo alongside the game project or reference via submodule.
